Шпаргалка по git

https://git-scm.com/book/ru/v1 - git book

Команда Что делает
git config --global user.name "name" Указываем имя
git config --global user.email "email" Указываем email
git init - инициализируем проэкт в папке
git status - проверяем статутс проэкта
git diff - предоставляет расширенную информацию об изменениях в еще не проиндексированных файлах
git diff --cached - предоставляет расширенную информацию об изменениях в проиндексированных файлах
git add . - индексируем содержимое нашего проэкта
git add index.html - индексируем конкретный файл нашего проэкта
git commit -m "comments" - добавляем под версионный контроль существующие файлы с добавлением коментария
git log - просмотр истории коммитов
git log -2 - просмотр последних 2 коммитов (число можна подставлять любое)
gitk - использование графического интерфейса для визуализации истории
git clone url - получить копию существующего репозитория Git
git remote add origin url - указываем путь к репозиторию
git push -u origin master - отправляем файлы на github
git checkout -b "new_branch" - создаем новую ветку с переходом на нее
git branch - показывает какие ветки у нас есть
